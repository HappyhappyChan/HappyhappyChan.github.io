2.2 二叉搜索树

 2022-04-24 2022-04-24 Coding / Algorithm 18 minutes read (About 2681 words)

Preface

Binary Search Tree, BST是一种特殊的二叉树，你只要记住他的两个主要特点：

  1. 左小右大，每个结点的左子树都比当前结点的值小，右子树都比当前结点的值大
  2. 中序遍历结果是有序的

这部分主要讲解BST的遍历、搜索、判断合法性、插入、删除、构造、后序遍历、快速排序。
